FAQ

Questions

Does it work on my existing PRs?

No — it activates when a PR is first opened (or reopened/converted from draft). It won't retroactively touch existing PRs.

What if I already wrote a description?

PR Autopilot checks the existing body before doing anything. If you've already written 3+ meaningful lines, it skips the PR entirely and stays out of your way.

Can I customize the format?

Custom templates are on the roadmap for Team plan users. For now, the output follows a consistent What / Why / How to test / Notes structure.

Does it read my entire codebase?

No. It only sees the diff that GitHub sends in the webhook payload — the actual lines changed in the PR. It never clones your repo or accesses history.

What model does it use?

Claude Haiku (Anthropic) — chosen for speed and accuracy. Each description takes about 2–5 seconds to generate.

How do I uninstall it?

GitHub App → Settings → Applications → Installed GitHub Apps → Configure → Remove. Takes 10 seconds. All generated descriptions stay (they're just text in your PRs).
